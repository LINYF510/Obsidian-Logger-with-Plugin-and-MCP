# Examples - 示例和模板

本目录包含 Obsidian Cursor Logger 的使用示例、工作流模板和配置示例。

## 目录结构

```
examples/
├── workflows/          # 工作流示例
│   ├── daily-report.md
│   ├── error-analysis.md
│   └── performance-check.md
├── plugin-examples/    # 插件开发示例
│   └── sample-plugin/
├── mcp-examples/       # MCP 使用示例
│   └── cursor-prompts.md
└── README.md          # 本文件
```

## 工作流示例

### workflows/daily-report.md
展示如何使用 MCP 工具生成每日开发报告：
- 日志统计
- 错误汇总
- 性能指标
- Auto-Reload 统计

### workflows/error-analysis.md
演示错误分析的完整流程：
- 获取最近错误
- 深度错误分析
- AI 智能建议
- 修复验证

### workflows/performance-check.md
性能检查工作流：
- CPU 和内存监控
- 日志吞吐量测试
- 响应时间分析
- 优化建议

## 插件开发示例

- 最佳实践和常见模式
- 调试技巧

## MCP 使用示例

## 使用指南

1. **开始开发前**：阅读 workflow 示例了解最佳实践
2. **遇到问题时**：参考 error-analysis.md 的分析流程
3. **学习插件开发**：查看 plugin-examples 中的示例代码
4. **提高效率**：使用 cursor-prompts.md 中的提示词模板
